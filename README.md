# water-cycle-match-game
An interactive, educational matching game built with **React** and **Tailwind CSS**. This application helps students learn and test their knowledge on key hydrological terms.

## Features
* **Dynamic Shuffling:** Terms and definitions are randomized every time the game resets.
* **Interactive UI:** Smooth transitions, active state highlights, and custom shake animations on incorrect matches.
* **Score & Move Tracking:** Keeps track of your current matches and total attempts.
* **Victory Screen:** Celebrates with a summary score once all 9 terms are correctly matched.

## Terms Covered
* Hydrological Cycle
* Precipitation, Condensation, Evaporation, Transpiration
* Surface run-off, Infiltration, Through flow
* Drainage basin

## Built With
* React (Hooks: `useState`, `useEffect`)
* Tailwind CSS (Styling & Animations)
* Lucide React (Icons)
